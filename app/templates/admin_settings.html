🎁 景品設定

スロットの点数に応じた等級と景品を設定できます。0点の景品を設定すると「ハズレなし」になります。

🎰 スロット確率設定

スロットのシンボルと配当、確率を設定できます。確率は自動調整されますが、手動でも調整可能です。

スロットを5回スピンしたときの平均合計点数です。
1回のスピンでハズレになる確率です。「確率を自動調整」を押すと、ハズレ確率と期待値に基づいて各シンボルが3つ揃う確率が自動計算されます。
{% for symbol in slot_config.symbols %} {% endfor %}
ID 表示 配当 抽選確率(%)* 色 不使用
* 抽選確率：ハズレでなかった場合の各シンボルの抽選確率です。実際の発生確率は「抽選確率 × (100% - ハズレ確率)」になります。
例：抽選確率10%、ハズレ確率20%の場合、実際の発生確率は 10% × 80% = 8% です。

📊 確率計算ツール

特定の点数範囲になる確率を計算します。

結果:

5回スピンで合計点数が指定範囲内になる確率です。
※ ハズレ確率も考慮された実際の発生確率です。

🏢 業種・AIコメント設定

業種を設定すると、AIが生成する口コミ文章をその業種に合わせた内容に調整します。追加の指示文でさらに細かくカスタマイズできます。

🤖 OpenAI API設定（アプリ個別）

このアプリ専用のOpenAI APIキーを設定できます。未設定の場合は店舗設定またはテナント設定のキーが使用されます。

APIキーは OpenAI Platform で取得できます。
